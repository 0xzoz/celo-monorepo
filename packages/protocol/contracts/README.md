# Protocols

Smart contracts for the Celo protocols, including identity and stability.

## Usage

```
RELEASE="celo-core-contracts-v2.mainnet"
URL="https://gitpkg.now.sh/celo-org/celo-monorepo/packages/protocol/contracts?$RELEASE"
npm install $URL
yarn add $URL
```
